Install Quality Assistant

A local website quality auditing tool that checks accessibility, visual, content, SEO, performance, and security issues — all in one click.

1Download the setup script
powershell -ExecutionPolicy Bypass -File setup.ps1
2Start the server
npm start
3Open your browser
http://localhost:3000
1Clone the repository
git clone https://github.com/urunkarpm/Quality-Assistant.git
cd Quality-Assistant
2Install dependencies
npm install
npx playwright install chromium
3Start the server
npm start
4Open your browser
http://localhost:3000

Settings Requirements

  • Node.js v18 or newer
  • npm v9 or newer
  • Windows: PowerShell (Administrator)
  • macOS/Linux: Git (optional, for cloning)
