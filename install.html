Install Quality Assistant

A powerful local website quality auditing tool that checks accessibility, visual design, content, SEO, performance, and security — all in one click.

1 Download & Run the Setup Script
powershell -ExecutionPolicy Bypass -File setup.ps1
2 Start the Server
npm start
3 Open Your Browser
http://localhost:3000
1 Clone the Repository
git clone https://github.com/urunkarpm/Quality-Assistant.git
cd Quality-Assistant
2 Install Dependencies
npm install
npx playwright install chromium
3 Start the Server
npm start
4 Open Your Browser
http://localhost:3000

System Requirements

  • Node.js v18 or newer
  • npm v9 or newer
  • PowerShell (Windows Administrator)
  • Git (macOS/Linux, optional)
